# VergeStore

VergeStore is a modern, high-performance e-commerce platform built with React, TypeScript, and Vite. Designed with a sleek, responsive UI and beautifully fluid animations, it provides everything you need to build your dream business online.

## Features

- **Modern Glassmorphism UI:** Clean gradients, blur backdrops, and modern aesthetic.
- **Dynamic Animations:** Interactive elements and smooth entering states powered by Framer Motion.
- **Cursor-Tracking Glow:** Dynamic background radial gradients that follow user cursor interactions.
- **Responsive Design:** Optimized for any screen size using Tailwind CSS.
- **Light & Dark Mode:** Full support for both themes using modern UI practices.

## Tech Stack

- [React](https://reactjs.org/) (with TypeScript)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React](https://lucide.dev/) (Icons)
- [React Router](https://reactrouter.com/)

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173` to view it in the browser.
